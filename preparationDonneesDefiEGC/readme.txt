Les données utilisées ont été construites en 3 temps :

Répertoire ListeArticles :
************************
Ce répertoire contient, pour chaque article des séries EGC, un fichier json 
contenant les informations extraites du fichier fourni par le défi 
complétées par le lieu de la conférence.

Répertoire ListeArticles_complement_villeAuteur :
***********************************************
Ce répertoire contient, pour chaque article, un fichier json contenant 
des informations sur la localisation de ses auteurs.

Répertoire ListeArticles_complement_metaSession :
***********************************************
Ce répertoire contient, pour chaque article, un fichier json contenant 
des informations sur la méta-session qui lui a été attribuée.
